# 🍳 Кулинарыч · Дашборд администратора

Локальный Streamlit-дашборд для просмотра базы данных бога (snapshot SQLite из Amvera).

## ⚡ Быстрый старт

```bash
# 1. Установи зависимости (в виртуальном окружении или глобально)
cd dashboard
pip install -r requirements.txt

# 2. Положи свежий дамп БД
#    Выгрузи /data/bot.db из Amvera → положи в dashboard/data/bot.db
cp /path/to/downloaded/bot.db data/bot.db

# 3. Запусти дашборд
streamlit run app.py
```

В браузере откроется `http://localhost:8501`.

## 📄 Страницы

### 🏠 Главная
Метрики: всего пользователей, групп, рецептов, ингредиентов.

### 📊 Обзор
- Рецепты по категориям (bar + таблица)
- Топ-10 групп по количеству рецептов
- Топ-15 ингредиентов
- Пользователи по количеству групп

### 👥 Группы
- Таблица всех групп с member/recipe count и типом (личная/общая)
- Выбор группы → участники + рецепты
- Распределение по типам

### 🍳 Рецепты
- Таблица всех рецептов с количеством групп и ингредиентов
- Фильтр по категории
- Выбор рецепта → в каких группах сохранён, ингредиенты, source URL

### 🔍 Поиск
- Поиск по ингредиенту (LIKE)
- Поиск по source URL
- Поиск по slug
- Свободный SQL-браузер (только `SELECT`/`WITH`)
- Подсказка по схеме БД

## 🔄 Обновление данных

Дашборд работает со **snapshot-ом** БД. Чтобы обновить данные:

1. Выгрузи свежий `bot.db` из Amvera (например, через веб-интерфейс Amvera)
2. Положи в `dashboard/data/bot.db` (замени старый)
3. Нажми **R** в браузере или кнопку **Rerun** в Streamlit

Подключение и кэши пересоздадутся автоматически.

## ⚙️ Конфигурация (опционально)

По умолчанию дашборд ищет БД в `dashboard/data/bot.db`. Если хочешь другой путь:

1. Скопируй `.streamlit/secrets.toml.example` → `.streamlit/secrets.toml`
2. Укажи нужный URL:

```toml
[connections.bot_db]
url = "sqlite:///C:/absolute/path/to/bot.db"
```

## 🗄 Схема БД

| Таблица | Колонки |
|---------|---------|
| `users` | `user_id`, `active_group` |
| `groups` | `group_id`, `name`, `owner_id`, `invite_code` |
| `group_members` | `group_id`, `user_id` |
| `group_recipes` | `group_id`, `category`, `slug` |

## 🛠 Технологии

- **Streamlit** — UI и навигация
- **Pandas** — таблицы и преобразования
- **Plotly** — интерактивные графики
- **sqlite3** (stdlib) — чтение БД

## ⚠️ Ограничения

- **Snapshot-режим**: данные устаревают между обновлениями
- **Нет `created_at`**: не покажем динамику по времени без миграции БД
- **Размер файла**: при росте БД > 50 МБ может быть неудобно передавать через мессенджеры — выгружай напрямую из volume Amvera
